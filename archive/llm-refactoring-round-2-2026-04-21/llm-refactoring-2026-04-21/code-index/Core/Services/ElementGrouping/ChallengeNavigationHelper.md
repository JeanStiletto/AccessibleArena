## Top-level comments
- Centralized helper for Direct Challenge / Friend Challenge screen navigation. Two-level navigation: ChallengeMain (spinners + buttons) and Deck Selection (folders). Polls for opponent join/leave, countdown start/cancel, and local player status changes every second. Uses reflection on UnifiedChallengeDisplay, ChallengePlayerDisplay, UnifiedChallengeBladeWidget.
